# 소재 주머니 룰렛 🎰

소재/질문을 랜덤으로 뽑아주는 재미있는 웹앱입니다.

## 주요 기능

- 🎲 랜덤 소재 뽑기
- ✨ 축하 애니메이션 (컨페티)
- 📝 소재 관리 (추가/수정/삭제)
- 💾 소재 데이터 JSON 파일로 관리

## 기술 스택

- React 19
- Vite
- React Router
- CSS3 (애니메이션)

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build
```
